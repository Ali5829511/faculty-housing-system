# هيكل موقع نظام إدارة المرور الجامعي

## البنية الأساسية

### الصفحات الرئيسية

#### 1. الصفحة الرئيسية (Home)
- شعار الجامعة والعنوان
- نظرة عامة على النظام
- الإحصائيات الرئيسية
- الوصول السريع للأقسام الرئيسية
- معلومات الجامعة والتواصل

#### 2. لوحة التحكم (Dashboard)
- ملخص الإحصائيات الرئيسية
- الرسوم البيانية والمخططات
- التنبيهات والإشعارات
- الوصول السريع للعمليات الشائعة
- آخر التحديثات والأنشطة

#### 3. قاعدة بيانات المركبات (Vehicles Database)
- جدول المركبات الشامل
- البحث والتصفية المتقدمة
- إضافة مركبة جديدة
- تعديل بيانات المركبة
- حذف المركبة
- عرض تفاصيل المركبة
- إدارة الصور والمستندات
- تسجيل الدخول والخروج
- الإحصائيات المفصلة

#### 4. قاعدة بيانات المخالفات (Violations Database)
- جدول المخالفات الشامل
- البحث والتصفية المتقدمة
- إضافة مخالفة جديدة
- إدارة الدفع والاعتراضات
- عرض تفاصيل المخالفة
- الإحصائيات والتقارير
- تتبع حالة الدفع

#### 5. مركز قواعد البيانات (Databases Hub)
- روابط سريعة لجميع قواعد البيانات
- إحصائيات عامة
- عمليات بحث موحدة
- تصدير البيانات

#### 6. التقارير والإحصائيات (Reports)
- تقارير المركبات
- تقارير المخالفات
- تقارير الدفع
- تقارير الأمان
- رسوم بيانية متقدمة

#### 7. الإعدادات (Settings)
- إعدادات النظام
- إعدادات المستخدم
- إعدادات الأمان
- إعدادات الإشعارات

## المكونات المشتركة

### الرأس (Header)
- شعار الجامعة
- اسم النظام
- قائمة التنقل الرئيسية
- أيقونات البحث والإشعارات
- قائمة المستخدم

### الشريط الجانبي (Sidebar)
- قائمة التنقل الرئيسية
- الروابط السريعة
- المعلومات الإضافية

### الفوتر (Footer)
- معلومات الجامعة
- الروابط المهمة
- معلومات التواصل
- حقوق النشر

## نقاط الاستجابة

### Desktop (> 1024px)
- عرض كامل مع الشريط الجانبي
- جداول مفصلة
- جميع الأعمدة مرئية

### Tablet (768px - 1024px)
- شريط جانبي قابل للطي
- جداول مختصرة
- بعض الأعمدة مخفية

### Mobile (< 768px)
- بدون شريط جانبي (قائمة هامبرجر)
- جداول أفقية قابلة للتمرير
- عرض مبسط
